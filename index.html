Bitpanda Transaction Explorer

Explore your blockchain deposits & withdrawals from Bitpanda

This tool fetches all blockchain deposits and withdrawals from your Bitpanda account, giving you insight into all your blockchain addresses. You'll need a read-only API key:

  1. Go to web.bitpanda.com/my-account/apikey
  2. Click "Create API Key" — enable only Transactions and Balances permissions
  3. Paste the key below and click Fetch Data
  4. Optionally click Save Key to persist it in your browser's local storage

Your API key is only used locally — it is never sent anywhere else.
